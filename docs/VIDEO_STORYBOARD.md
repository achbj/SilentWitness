## Production Notes

### Recording the Demo
1. **Use Expo web preview** or **screen recording** of actual app
2. **Record in segments** for easier editing
3. **Use dark theme** for dramatic effect

### Key Shots to Capture
```
Shot 1: Decoy calculator (before unlock)
Shot 2: Unlock transformation
Shot 3: Recording in progress
Shot 4: Classification result
Shot 5: Legal document preview
Shot 6: Emergency delete (hover, don't click)
Shot 7: Offline status badge
Shot 8: Final logo/title card
```

### Text Overlays Template
```
Title font: Sans-serif, bold
Body font: Clean, readable
Colors:
  - Primary: #2196F3 (blue)
  - Danger: #F44336 (red)
  - Safe: #4CAF50 (green)
  - Background: #212121 (dark)
```

### Animation Guidelines
```
Transitions: Fade (0.5s)
Text reveal: Slide up or fade in
Icons: Subtle pulse or glow
Avoid: Flashy, distracting effects
```
